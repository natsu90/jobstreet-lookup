
## WhatstheSalary-Web

Running [WhatstheSalary](https://github.com/natsu90/whatsthesalary) from web

### Installation

```
npm install

node node_modules/puppeteer/install.js

npm run start
```

Open [localhost:3002](http://localhost:3002/) from your browser
